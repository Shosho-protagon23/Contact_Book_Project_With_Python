# Contact_Book_Project_With_Python
Aplikasi Buku Kontak desktop berbasis Python &amp; Tkinter. Fitur lengkap: CRUD (Tambah/Edit/Hapus), pencarian real-time, sistem pin kontak favorit, dan ganti tema warna (Dark Mode, dll). Data tersimpan portabel menggunakan JSON tanpa database server. Ringan dan mudah digunakan.
